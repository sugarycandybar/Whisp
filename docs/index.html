Built for Speed. Designed for GNOME.

Whisp abandons traditional file hierarchies in favor of a spatial, swipeable canvas.

gestures

Spatial Navigation

Fluidly swipe between your recent notes using 1:1 touchpad gestures via native Adwaita Carousels.

markdown

WYSIWYG Markdown

Real-time rendering of Markdown. Toggle WYSIWYG mode to instantly hide syntax symbols and view clean rich text.

themes

Paper Themes

Native dynamic styling. Choose between Dotted, Grid, or Blank backgrounds to mimic physical scratchpads.

link

Smart Paste

Paste a URL to automatically shrink it via TinyURL in the background, or use plain-text paste to strip source formatting.

The Road Ahead

We are actively building Phase 2 of Whisp: The Reactive Math Engine.

Phase 1

Core Editor & Polish

Paper themes, smart text extraction, URL shortening, and Flathub packaging.

Phase 2

Reactive Math Engine

A contextual evaluator that processes arithmetic and variables line-by-line, overlaying ghost-text results in the margin.

Phase 3

Advanced Context

Unit conversions, advanced variable references across notes, and OCR extraction modules.
