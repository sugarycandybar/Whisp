Spatial Navigation

Whisp abandons traditional file hierarchies. Fluidly swipe between your recent notes using 1:1 touchpad gestures natively powered by Adwaita Carousel.

WYSIWYG Markdown

Experience real-time rendering of your Markdown syntax. Toggle WYSIWYG mode on to instantly hide the Markdown symbols and view clean, distraction-free rich text.

Smart Link Extraction

Pasting a URL automatically fetches a shortened TinyURL in the background so your notes stay clean. Need plain text? Standard extraction aggressively strips rich-text formatting on paste.

Under the Hood

Paper Themes

Native dynamic CSS styling. Toggle between Dotted, Grid, or Blank backgrounds to mimic physical engineering paper.

Keyboard Centric

Never leave your keyboard. Instant shortcuts for formatting (`Ctrl+B/I/U`), lists, and instant URL shortening (`Ctrl+Shift+L`).

Lightning Fast

Whisp maintains a highly optimized footprint by exclusively rendering your active working set, ensuring instant startup times.

Coming Next: Math Engine

Phase 2 is actively developing a contextual evaluator to process arithmetic line-by-line and overlay ghost-text results in the margin.
