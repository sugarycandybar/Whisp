The Anti-Note for GNOME.

A fluid, gesture-driven scratchpad designed for absolute speed.

Download on Flathub (Soon)

Core Features

Spatial Navigation

Whisp abandons traditional file hierarchies. Fluidly swipe between your recent notes using 1:1 touchpad gestures natively powered by Adwaita Carousel.

WYSIWYG Markdown

Toggle WYSIWYG mode on to instantly hide Markdown symbols and view clean rich text.

Smart Extraction

Paste a URL and it automatically shrinks via TinyURL in the background. Pure text extraction strips formatting.

Native Paper Themes

Select between Dotted, Grid, or Blank backgrounds to perfectly mimic the feel of a physical engineering notebook right on your desktop.

Coming Soon: The Math Engine

Phase 2 brings a contextual evaluator that processes arithmetic line-by-line and overlays ghost-text results directly in the margins of your notes.
